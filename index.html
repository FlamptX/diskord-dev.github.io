About Diskord

A powerful API wrapper around Discord API written in Python.

Diskord is originally a fork of discord.py by Rapptz which was discontinued in August 2021. The purpose of this library is to provide full coverage of Discord API with an easy to use interface to aid in bot development by abstracting away the complicated steps.

Features

  • Modern API design using async and await.
  • Proper ratelimit handling.
  • Optimised in both speed and memory.
  • Actively maintained.
  • Implements the entire Discord API.
  • Easy to use object oriented interface.
